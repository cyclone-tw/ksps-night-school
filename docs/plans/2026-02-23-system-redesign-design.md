# 國姓國小進修部管理系統 — 改版設計文件

> 日期：2026-02-23
> 狀態：已核准

---

## 1. 專案概述

將現有的國姓國小進修部管理系統從「寫死資料的靜態前端」改版為「資料動態化 + 權限分流 + 薪資自動化 + 報表優化」的完整管理平台。

### 核心目標

1. **資料動態化** — 教師/學生/課程/費用設定全部從 Google Sheet 載入，不寫死在程式碼中
2. **權限分流** — 教師（公開）只能填報；管理者（密碼）可查看全部功能和報表
3. **薪資自動化** — 根據教學日誌自動計算鐘點費，生成月薪資總表 + 個別薪資條 XLS
4. **教學日誌 XLS 排版優化** — A4 大小、放大字體、簽名欄加大、簽章改為進修部主任+校長
5. **出缺席報表** — 自訂日期區間 + 勾選學生
6. **成績管理** — 預留架構，未來實作

## 4. 前端權限與頁面結構

僅顯示：
- 📝 教學日誌填報
- 📋 學生出缺席填報

### 4.2 管理者模式（密碼解鎖）

頁面上有「管理者登入」按鈕，輸入密碼後解鎖完整功能：
- 📊 總覽儀表板（今日出缺席率 + 今日教學日誌）

- 📋 學生出缺席填報
- 📥 報表下載中心
- 📚 成績管理（灰色，開發中）

- 密碼存在 Google Sheet `系統設定` 分頁
- 前端向 GAS 發送驗證請求，GAS 端比對
- 驗證成功後用 `sessionStorage` 保存狀態，關閉瀏覽器即失效

## 5. Google Apps Script API 設計

### 5.1 公開 API（不需密碼）

| API | 方法 | 說明 |
|-----|------|------|
| `load_config` | GET | 載入教師/學生/課程名單（不含費用金額、密碼） |
| `submit_log` | POST | 提交教學日誌 |
| `submit_attendance` | POST | 提交出缺席記錄 |

### 5.2 管理者 API（需密碼驗證）

| API | 方法 | 說明 |
|-----|------|------|
| `verify_admin` | GET | 驗證管理者密碼 |
| `load_admin_config` | GET | 載入完整設定（含費用、統計） |
| `get_dashboard` | GET | 今日出缺席率 + 今日教學日誌 |
| `export_log` | GET | 生成教學日誌 XLS，回傳下載連結 |
| `export_salary` | GET | 生成月薪資總表 XLS，回傳下載連結 |
| `export_payslip` | GET | 生成薪資條 XLS，回傳下載連結 |
| `get_students_in_range` | GET | 查詢日期區間內有記錄的學生列表 |
| `export_attendance` | GET | 生成出缺席報表 XLS，回傳下載連結 |

### 5.3 API 資料格式

**load_config 回傳：**
```json
{
  "schoolName": "國姓國民小學",
  "classTime": "19:00~21:00",
  "staff": [
    { "name": "吳怡萱", "role": "導師" },
    { "name": "劉政勳", "role": "教師" }
  ],
  "students": [
    { "name": "阮氏彫", "status": "在學" }
  ],
  "courses": [
    { "name": "國語與彈性", "weekday": "一", "teacher": "吳怡萱" }
  ]
}
```

**submit_log POST body：**
```json
{
  "date": "2025-11-03",
  "weekday": "一",
  "time": "19:00~21:00",
  "course": "國語與彈性",
  "content": "第三課均衡飲食字詞練習",
  "teacher": "吳怡萱"
}
```

**submit_attendance POST body：**
```json
{
  "date": "2025-11-03",
  "weekday": "一",
  "course": "國語與彈性",
  "attendance": {
    "阮氏彫": "✓",
    "阮紅妮": "✓",
    "范宥嫺": "△"
  }
}
```

**XLS 生成流程：**
```
前端選參數 → 呼叫 API（帶密碼）
→ GAS 讀取資料 Sheet，篩選條件
→ 用 Spreadsheet API 建立新 XLS 檔案
→ 套用格式（標楷體、欄寬、列高、合併、框線）
→ 回傳 { success: true, downloadUrl: "...", fileName: "..." }
→ 前端開啟下載連結
```

## 6. XLS 報表規格

### 6.1 教學日誌 XLS

- **方向**：A4 橫向
- **字體**：全部標楷體
- **標題**：`南投縣國姓國小進修部 {民國年}年度 {月}月 教學日誌`，20pt 粗體
- **表頭**：序號、日期、星期、時間、課程、上課內容、教師簽名、授課教師，14pt 粗體
- **內容**：14pt
- **列高**：55pt（確保簽名空間）
- **教師簽名欄寬**：32（原 26，加大方便簽名）
- **簽章區**：`進修部主任：` `校長：`，18pt 粗體
- **不含**承辦、出納、會計

### 6.2 月薪資總表 XLS

- **標題**：`國姓國民小學進修部 {民國年}年{月}月支給費用`，標楷體 16pt 粗體
- **表頭**：姓名、上課期間、授課天數、授課節數、鐘點費單價、合計、額外費用、合計、備註，14pt 粗體
- **人員排列**：校長排第一（鐘點欄留空，只有額外費用）、其餘教師依序
- **自動計算**：
  - 授課天數 = 從教學日誌 Sheet 篩選該月該教師記錄數
  - 授課節數 = 天數 × 3
  - 鐘點費合計 = 節數 × 單價
  - 額外費用 = 從人員名冊讀取
  - 總合計 = 鐘點費合計 + 額外費用
- **備註欄**：自動列出所有授課日期（所有教師都有，校長留空）
- **合計列**：各欄加總
- **簽章區**：承辦、出納、會計、校長

### 6.3 薪資條 XLS

每人一個區塊，全部在同一個 Sheet：

- **校長**：標題「...校長兼職費」，欄位：姓名、上課期間、校長兼職費、合計、備註
- **有額外費用的教師**：標題「...鐘點費與{費用名稱}」，欄位：姓名、上課期間、天數、節數、單價、合計、{費用名稱}、合計、備註
- **純鐘點教師**：標題「...鐘點費」，欄位：姓名、上課期間、天數、節數、單價、合計、備註
- **統一格式**：沒有的欄位留空
- 字體：標楷體 13pt

### 6.4 出缺席報表 XLS

- **操作流程**：選日期起迄 → 勾選學生 → 生成
- **標題**：`南投縣國姓國小進修部 出缺席記錄表（{起}~{迄}）`
- **欄位**：姓名 | 各上課日（含星期）| 出席天數 | 出席率 %
- **記號**：✓ 出席、△ 請假、空白 未記錄

## 8. 非功能需求

- **部署**：前端 GitHub Pages，後端 GAS Web App
- **瀏覽器**：Chrome、Edge、Safari（現代瀏覽器）
- **手機**：響應式設計，手機可填報
- **效能**：GAS 6 分鐘執行限制，以此資料量不是問題
- **成本**：零成本（GitHub Pages + Google 免費服務）
